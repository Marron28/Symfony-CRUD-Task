                            SYMFONY 

Objective: Build a simple Task Manager Web App Using:

        - Symfony (Latest Version)

        - Doctrine ORM (Database)

        - Twig (Templating)

        - TailwindCSS (Styling)

Features:

1. Create Task Entity with fields:

        - id (Primary Key)

        - title (String, Required)

        - Description (String, Optional)

        - Status(Enum/string: pending, in_progress, done)

        - created_at (Date Time)

2. Crud Functionality:

        - Create Task:

                - Form to add new task

                - Validate Required fields

        - Read Task:

                - List all Tasks in table

                - Show: title, status, created date, actions(Edit/Delete)

        - Update Task:

                - Edit Form with pre-filled values

        - Delete Task

                - Delete button with confirmation

3. Create UI:

        -Use tailwind

                - Table layout

                - Styled button (Add, Edit, Delete)

                - Form Styling

                - Status Badges:

                        - Pending: gray

                        - In progress: yellow

                        - Done: Green

- Create and run your own database migration

- Add readme file for setup instruction

- Push your project to your GitHub repo.
